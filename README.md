IPL Cricket Data Analysis

This project focuses on analyzing IPL (Indian Premier League) cricket match data using Python and data visualization techniques. The aim is to identify team performances, toss winning patterns, player statistics, and venue analysis using IPL datasets.

Objectives
- Perform data preprocessing and cleaning
- Analyze IPL team performances
- Analyze toss winning patterns
- Identify top performing players
- Generate meaningful cricket insights
- Create visualizations using graphs

Dataset Features
- Match ID
- Season
- Team Names
- Toss Winner
- Match Winner
- Player of the Match
- Venue
- Umpires
- Match Date

Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
